## Top remaining failure signatures (eras C+D)

1. **craftRecipe 51% fail (209 uses).** 63× `TypeError: Cannot read
   properties of null (reading 'length')` — `getItemCraftingRecipes()`
   returns null for items with no crafting recipe (charcoal, etc.) and
   `craftRecipe` calls `.length` on it: a crash presented as a failure.
   38× `missing ingredient` thrown by `bot.craft` with no detail when
   craftable-count is zero.
2. **smeltItem 74% fail (85 uses).** Bots try to smelt `coal`/`coal_ore`/
   `raw_coal` chasing charcoal; the "smelt the raw item" hint sends them in
   circles. They never learn charcoal = smelt logs.
3. **givePlayer 50% fail (58 uses).** Two classes: receiver never picks up
   the discarded items (busy mid-action), and giver doesn't have the item
   at all. Chest transfer sidesteps class 1; clearer inventory errors fix 2.
4. **Site discipline collapsed overnight**: 61% of D_clean steps underground
   (y<75), 21% >30 blocks from site. Journal GOAL lines only 64% on-mission
   in the recent quarter (samples: "Fulfill Sable's crafting orders",
   "Extend bridge southward"). The site anchor holds *coordinates* but the
   mining pull invents material side quests (copper!) that the mission
   never asked for.
5. **Query churn**: !nearbyBlocks 0.8% → 11.4% of steps — mostly confusion
   downstream of craft/smelt failures.
6. **Chat is 1% of steps** and half of those are outage sentinels. Fine for
   reliability data; a real gap for personality-bearing teacher data.
   Deliberately deferred until after the clean run baseline.

## Edit wave for the clean run (E1–E7)

- **E1 craftRecipe**: null-guard (explain "not craftable, comes from
  smelting/mobs/mining") + pre-check the limiting resource and report
  "need N more X (have A, need B)" instead of throwing.
- **E2 smeltItem**: intent-aware hints — coal/coal_ore/raw_coal get
  "charcoal = smelt logs (!smeltItem('oak_log'))"; keep raw-hint otherwise.
- **E3 chest-first cooperation**: prompt rule — material transfers go
  through the site chest (!putInChest / !takeFromChest + announce);
  !givePlayer only for tools when partner is idle beside you. Site seed
  places a chest.
- **E4 depth/mission guard**: code nudge — persistent y<75 or >40 blocks
  from site injects a system reminder that mining is only for mission
  materials and to resurface with the haul.
- **E5 journal GOAL tether**: GOAL line must restate the standing mission
  phase, never an invented objective; side projects only after the farm
  exists.
- **E6 pair resync**: on partner (re)connect, inject a one-shot system
  note to re-sync roles (fixes "Jolt is idle/unresponsive" drift after
  restarts).
- **E7**: keep loop-breaker v2, near-arrival, collect/craft name guards,
  memguard, watchdog v4 unchanged.

## Clean-run protocol

New world (chosen seed: plains + wood + water), fresh site anchor, chest +
crafting table seeded at site, journals seeded with role split, archived
dev-era episodes (`episodes_dev/`), run manifest recording config hashes,
2-bot roster (cooperation is the data moat; drop to 1 only if coordination
failures dominate day one despite E3/E6).
